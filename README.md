# Personal Portfolio (GitHub Pages)

Static portfolio site for I Dewa Gde Widhy Suryana. Built with HTML, Tailwind CSS, and vanilla JavaScript. Content is data-driven from JSON files under `data/`.

## Structure
- `index.html` — layout and sections
- `main.js` — rendering logic + UI behaviors
- `tailwind.css` — compiled styles + custom overrides
- `data/` — content (education, experiences, projects, skills)
- `images/` — profile photo, logos, project images

## Content Updates
Edit the JSON files in `data/`:
- `data/education.json`
- `data/experiences.json`
- `data/projects.json`
- `data/skills.json`

## Contact Form
The contact form uses a simple mailto fallback (no backend). Submissions open the visitor’s email client.

## Local Preview
Open `index.html` in a browser.

## Notes
This site is static and GitHub Pages–friendly; all assets are public.
